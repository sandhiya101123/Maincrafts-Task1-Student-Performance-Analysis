# Student Performance Analysis

This is my Task 1 submission for the Data Science and Data Analysis with Python Internship.

In this project, I analyzed the Student Performance dataset using Python. I checked the dataset, handled data cleaning, calculated 
some basic statistics, and created graphs to understand student performance.

### Tools Used

* Python
* Pandas
* Matplotlib
* Google Colab

### Findings

* Average Final Grade: 10.42
* Students Scoring Above 15: 40
* Study Time and Grade Correlation: 0.0978
* Female Average Grade: 9.97
* Male Average Grade: 10.91
